🔄 Synchroniser les données entre appareils
Les articles, membres et paramètres sont stockés dans chaque navigateur séparément. Pour les transférer d'un appareil à un autre (PC → téléphone, etc.), exportez puis importez le fichier.
⚠️ Important
Un article créé sur votre PC n'apparaît pas automatiquement sur votre téléphone (et inversement). Utilisez l'export/import ci-dessous pour synchroniser les deux appareils.

📤 Exporter mes données

Télécharge un fichier slf-backup.json qui contient tous vos articles et membres. Conservez-le pour transférer sur un autre appareil ou pour faire une sauvegarde.

📥 Importer une sauvegarde

Charge un fichier slf-backup.json précédemment exporté. Vous pouvez choisir d'ajouter les données ou de remplacer celles déjà présentes sur cet appareil.

🗑 Réinitialiser les données

Supprime tous les articles et membres stockés sur cet appareil uniquement. La session admin sera conservée.
